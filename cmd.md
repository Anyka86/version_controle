## Подсказка по командной строке

Команда смены дирректории 
```sh
cd c:\folder_name
```

Команда отображения текущей дерриктории MacOs, Linux
```sh
pwd
```
Листинг текущей директории 
Windows:
```sh
dir
```
Linux, MacOs:
```sh
ls
````

Удаление файла Windows:

```sh
del<filename>
```

в Linux, MacOs:
```sh
rm<filename>
```
